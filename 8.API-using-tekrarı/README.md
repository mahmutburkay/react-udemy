# 🚀 React + JSON Server Projesi

Bu proje, React (Vite) frontend ve `json-server` ile simüle edilmiş bir API yapısını kullanmaktadır.

## 🛠️ Kurulum

Bağımlılıkları yüklemek için terminale yazın:

```bash
npm install

```

Çalıştırma
Projenin çalışması için iki ayrı terminal sekmesi kullanın:

1. API (Backend)
   Terminalde şu komutu çalıştırarak veri tabanını 3005 portunda ayağa kaldırın:

   ```bash
   npx json-server src/db.json --port 3005
   ```

```
npx json-server src/db.json --port 3005
```

2. Frontend (React)
   Diğer terminal sekmesinde React uygulamasını başlatın:
   ```
    npm run dev
   ```
